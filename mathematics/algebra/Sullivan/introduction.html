Real Numbers

We can denote a set via the roster method or the set-builder notation.

Each time we expand a number system, we do so in order to be able to handle new, and usually more complicated, problems.

A rational number is a number that can be expressed as a qutient a/b of two integers, where b ≠ 0.

Rational numbers may be represented as decimals.

Every rational number may be represented by a decimal that either terminates or is nonterminating with a repeating block of digits, and vice versa.

Every irrational number may be represented by a decimal that neither repeats nor terminates.

Together, the rational numbers and irrational numbers form the set of real numbers.

Every decimal may be represented by a real number, and every real number may be represented by a decimal.

Truncation (correct to a given number of decimal places) - drop all the digits that follow the specified final digit in the decimal.

Rouding - Identify the specified final difit in the deciaml. If the next digit is 5 or more, add 1 to the final digit, otherwite leave the final digit as it is. Then truncate following the final digit.

Properties of Real Numbers

The four important properties of equality are listed next. b and c represent real numbers.

  1. The reflexive property states that a number always equals itself; i.e., a = a.
  2. The symmetric property states taht if a = b then b = a.
  3. The transitive property states that if a = b and b = c then a = c.
  4. The principle of substitution states that if a = b and b = c then we may substitute b for a in any expression containing a.

Other properties

The commutative property states that the order in which addition or multiplication takes place will not affect the final result. a + b = b + a

The associative property states that the way we add or multiply three real numbers will not affect the final result. a + (b + c) = (a + b) + c = a + b + c

The distributive property states that multiplying the sum of two or more addends by a number will give the same result as multiplying each addend individually by the number and then adding the products together. a ⋅ (b + c) = a ⋅ b + a ⋅ c (a + b) ⋅ c = a ⋅ c + b ⋅ c
